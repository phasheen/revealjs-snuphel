Development

Setting Up Your Environment

  • Create a NodeJS Workspace in Coder
  • Clone the template repository
  • Install dependencies with npm install
  • Start the dev server with npm start
  • Edit files in your preferred code editor
  • Changes appear instantly in browser

# Clone the repository
git clone https://github.com/phasheen/revealjs-snuphel

# Navigate to the directory
cd revealjs-snuphel

# Install dependencies
npm install

# Start the development server
npm start
Setting up your development environment is straightforward. The template uses npm for package management and includes a development server that automatically refreshes as you make changes.

Basic Slide Structure

  • Each slide is a <section> element
  • Vertical slides use nested sections
  • Use HTML or Markdown for content
  • Add notes with <aside class="notes">
<section>
  <h2>Slide Title</h2>
  <ul>
    <li>Bullet point 1</li>
    <li>Bullet point 2</li>
  </ul>
  <aside class="notes">
    Speaker notes here
  </aside>
</section>
The basic structure of a Reveal.js presentation is simple: each slide is a section element, and you can create vertical slides by nesting sections within each other.

Customizing Your Presentation

  • Edit custom/custom-scripts.js to update presenter info
  • Change theme color in custom/custom-styles.css
  • Add your own custom components
  • Modify slide transitions and animations
// In custom-scripts.js
const presenterName = "Your Name";
const presenterDepartment = "Your Department";
const presenterUniversity = "Your University";
const presenterEmail = "your.email@example.com";
const sessionName = "Your Presentation Title";
/* In custom-styles.css */
:root {
  --theme-color: #2a76dd; /* Change this color */
  --theme-color-transparent-10: rgba(42, 118, 221, 0.1);
}
Customizing the template is easy - just update your personal information in the JavaScript file and adjust the theme color in the CSS variables. For more advanced customization, you can modify any part of the CSS or JavaScript.

Deployment with GitHub & Cloudflare Pages

Why GitHub & Cloudflare?

  • Free hosting for Code and websites
  • Global CDN for fast loading worldwide
  • Automatic builds from your repository
  • Custom domains with SSL certificates
  • Version control for all changes

GitHub repository connected to Cloudflare Pages

https://github.com/
https://pages.cloudflare.com/
https://bejamas.com/compare/cloudflare-pages-vs-github-pages
GitHub and Cloudflare Pages offer an excellent free solution for hosting academic presentations, with automatic deployments, version control, and a global CDN.

GitHub Setup

  1. Create a new GitHub repository
  2. Push your presentation files to the repository
  3. Ensure your repository is public (or use GitHub Pro)
  4. Update your repository settings to allow Pages deployment

# Initialize git in your project folder
git init

# Add all files
git add .

# Commit the changes
git commit -m "Initial commit of my presentation"

# Add remote repository
git remote add origin https://github.com/username/my-presentation.git

# Push to GitHub
git push -u origin main
Setting up your presentation in GitHub is straightforward, especially if you're already familiar with Git. The key is to have a well-organized repository that includes all necessary files.

Cloudflare Pages Setup

  1. Sign up for a Cloudflare account
  2. Go to Workers and Pages->Create a page
  3. Connect your GitHub account
  4. Select your repository
  5. Deploy!

Cloudflare Pages project configuration

Cloudflare Pages provides a simple interface for connecting to your GitHub repository and automatically deploying your presentation whenever you push changes.

Custom Domain Setup

  1. In Cloudflare Pages, go to your project settings
  2. Select "Custom domains" tab
  3. Add your domain (e.g., slides.yourdomain.com)
  4. Configure DNS records
  5. Automatic SSL certificate provisioning
  6. Wait for propagation (~24 hours)

# Example DNS configuration
Type    Name               Content
CNAME   slides             pages.cloudflare.com

Example DNS configuration for custom domain

Adding a custom domain makes your presentation look more professional and easier to remember. Cloudflare handles SSL certificates automatically for secure connections.
